# CinematicCard — Animated Greeting Cards API

> **Your first card is completely free.** Music plays. Their name appears in calligraphy. Fireworks explode. No app required.

[cinematiccard.com](https://cinematiccard.com) — The digital greeting card platform that turns your message into a 60-second cinematic film.

## What is CinematicCard?

CinematicCard is an animated greeting card platform. Recipients tap a shareable link and experience:

1. Their name appearing in animated calligraphy as music starts
2. Cinematic fireworks, confetti, or falling petals
3. Your personal message revealing word-by-word
4. A photo slideshow (up to 20 photos)
5. Optional cash gift reveal via Venmo, PayPal, or CashApp

No app download. No account required. Link never expires.

## Agent API

CinematicCard supports AI agent purchases via the ACP (Agent Commerce Protocol).

- **OpenAPI spec**: `GET https://cinematiccard.com/api/agent/v1/openapi.json`
- **Agent manifest**: `GET https://cinematiccard.com/api/agent/v1/manifest`
- **Agent landing**: [cinematiccard.com/agents](https://cinematiccard.com/agents)
- **ACP discovery**: `GET https://cinematiccard.com/.well-known/agent-purchase.json`
